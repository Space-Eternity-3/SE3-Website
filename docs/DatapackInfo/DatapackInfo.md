Compiler working:
Text A - TextA

-------------------------------
You can't use these chars in variable names and values:
{ } < : ; ~ '
All other chars are allowed.
-------------------------------
> < - comments
{ } - variable tree operators
 :  - variable/value separator
 ;  - value end char
 '  - on/off remove space
-------------------------------
